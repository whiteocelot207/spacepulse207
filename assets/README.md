# Asset Library

This folder contains visual assets for the video renderer.

## Required Assets (Download and add these)

### Backgrounds
- `space_dark.jpg` - Dark starfield background (1080x1920)
- `space_nebula.jpg` - Colorful nebula background (1080x1920)
- `space_stars.jpg` - Dense star field (1080x1920)

### Planets (PNG with transparency)
- `earth.png` - Earth image
- `moon.png` - Moon image
- `sun.png` - Sun image
- `mars.png` - Mars image
- `jupiter.png` - Jupiter image
- `saturn.png` - Saturn with rings
- `venus.png` - Venus image
- `neptune.png` - Neptune image

## Free Asset Sources
- NASA Image Gallery: https://images.nasa.gov
- Unsplash (space): https://unsplash.com/s/photos/space
- Pixabay: https://pixabay.com/images/search/planet/

All NASA images are public domain and free to use.
